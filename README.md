# New Cerkinfo website


### requirements

`npm`

### install dev instance on your laptop:
1. download the repo
2. download `npm`
3. go into the project directory
4. run `npm install`
5. run `npm start`
6. It's functionnal! (if not, we are really sorry :'( )
